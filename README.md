# 底稿整理助手 v1.0

一个纯本地运行的 Word 报告格式调整工具，支持模板导入、管理、应用，一键调整报告格式。

## 功能特点

- **模板管理**：支持从现有 Word 文档导入格式模板，保存为模板文件
- **格式调整**：支持封面、标题、正文、表格、图片、页眉页脚、目录、落款等格式设置
- **一键应用**：选择模板后，一键应用到目标报告文件
- **自动备份**：应用模板前自动备份原文件，防止数据丢失
- **纯本地运行**：无需联网，所有操作在本地完成

## 支持的格式设置

### 封面设置
- 自动生成封面（若报告没有封面页）
- 报告标题字体、字号、加粗、对齐方式
- 单位名称字体、字号
- 日期字体、字号

### 标题设置
- 一级标题：字体、字号、加粗、对齐、段前距、段后距
- 二级标题：字体、字号、加粗、对齐、段前距、段后距
- 三级标题：字体、字号、加粗、对齐、段前距、段后距

### 正文设置
- 字体、字号、加粗、对齐方式
- 行距、首行缩进

### 表格设置
- 表头文字加粗
- 表头添加灰色底纹
- 单元格对齐方式
- 自动调整表格宽度

### 图片设置
- 图片居中对齐
- 最大宽度设置

### 页眉页脚设置
- 页眉内容、字体、字号、对齐方式
- 首页不显示页眉
- 页码格式（多种格式可选）
- 页码位置、起始页码、首页显示设置

### 目录和落款设置
- 插入目录页
- 落款（盖章/日期）右对齐
- 包含日期、日期格式

## 安装使用

### 环境要求
- Python 3.8+
- tkinter（Python 内置 GUI 库）
- python-docx 库

### 安装步骤

1. 克隆仓库：
```bash
git clone https://github.com/ztywudi/docx-format-tool.git
cd docx-format-tool
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行程序：
```bash
python format_tool.py
```

### 使用说明

1. **打开报告**：点击"打开报告"按钮，选择要调整的 Word 文档（.docx）
2. **导入模板**：点击"导入模板"按钮，可以从现有文档导入格式，或手动设置格式
3. **保存模板**：设置好格式后，点击"保存为模板"按钮，保存为模板文件
4. **应用模板**：选择要应用的模板，点击"一键应用模板"按钮
5. **查看结果**：在 Word 中打开调整后的文档，按 Ctrl+A → F9 更新目录和页码

## 模板管理

- **导入模板**：从现有 Word 文档导入格式设置
- **保存为模板**：将当前设置保存为模板文件
- **恢复默认**：恢复为内置默认模板
- **删除模板**：删除选中的模板文件

模板按分类管理，支持的分类有：
- 审计报告
- 税审报告
- 投标文件
- 通用模板
- 其他

## 技术栈

- **Python 3.8+**
- **tkinter**：GUI 界面
- **python-docx**：Word 文档处理
- **json**：模板文件存储

## 注意事项

1. 本工具只支持 .docx 格式，不支持 .doc 格式
2. 应用模板前会自动备份原文件，备份文件在原文件同目录下
3. 目录和页码需要在 Word 中手动更新（Ctrl+A → F9）
4. 模板文件保存在用户数据目录下（Windows：%APPDATA%\.底稿整理助手，Mac/Linux：~/.底稿整理助手）
