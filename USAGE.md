## 1. 软件简介

**底稿整理助手**是一款纯本地运行的 Word 报告格式调整工具，专为审计报告、税审报告、投标文件等场景设计。

**核心特点：**
- 📁 **纯本地运行**：无需联网，所有操作在本地完成，数据不外传
- 🎨 **模板管理**：可从现有文档导入格式，保存为模板反复使用
- 🚀 **一键应用**：选择模板后，一键调整报告格式
- 💾 **自动备份**：应用模板前自动备份原文件，防止数据丢失
- 📋 **分类管理**：按审计报告、税审报告、投标文件等分类管理模板

**支持的格式调整项：**

| 分类 | 可设置项 |
|------|---------|
| 封面 | 自动生成、标题字体/字号/加粗/对齐、单位名称、日期 |
| 标题 | 一/二/三级标题：字体、字号、加粗、对齐、段前距、段后距 |
| 正文 | 字体、字号、加粗、对齐、行距、首行缩进 |
| 表格 | 表头加粗、表头底纹、单元格对齐、自动调整宽度 |
| 图片 | 居中对齐、最大宽度限制（自动缩放） |
| 页眉/页码 | 页眉内容/字体/对齐、首页设置、页码格式/位置/起始页码 |
| 目录/落款 | 插入目录页、落款右对齐、日期格式 |

## 2. 安装方法

### 方法一：Windows 用户直接使用 exe（推荐）

1. 前往 [Release 页面](https://github.com/ztywudi/docx-format-tool/releases) 下载最新版 `底稿整理助手.exe`
2. 双击运行即可，**无需安装 Python**

> ⚠️ 首次运行时，Windows 可能弹出"无法识别的应用"提示，点击"更多信息" → "仍要运行"即可。

### 方法二：从源代码运行（所有系统）

**环境要求：** Python 3.8+

**步骤：**

```
第1步：克隆仓库
git clone https://github.com/ztywudi/docx-format-tool.git

第2步：进入项目目录
cd docx-format-tool

第3步：安装依赖
pip install -r requirements.txt

第4步：运行程序
python format_tool.py
```

### 方法三：pip 安装

```
pip install docx_format_tool-1.1.0-py3-none-any.whl
format-tool
```

## 3. 快速上手

### 三步完成格式调整

```
第1步：打开报告
  点击「📂 打开报告」→ 选择要调整的 .docx 文件

第2步：设置格式
  在右侧各标签页中调整格式参数
  或从左侧「导入模板」直接加载已有格式

第3步：一键应用
  点击「🚀 一键应用模板」→ 确认 → 完成！
```

### 工作流程图

```
打开报告 → 选择/设置模板 → 一键应用 → 在Word中更新目录和页码
  │              │               │              │
  │              │               │              └─ Ctrl+A → F9
  │              │               └─ 自动备份原文件
  │              └─ 导入模板 / 手动设置 / 恢复默认
  └─ 支持的格式：.docx
```

## 4. 功能详解

### 4.1 文件操作

| 按钮 | 功能 |
|------|------|
| 📂 打开报告 | 选择要调整格式的 Word 文档（.docx） |
| 📋 文件信息 | 查看当前文档的结构信息（标题数量、正文段落数、表格数） |

### 4.2 封面设置

- **☑ 自动生成封面**：勾选后，如果报告没有封面页，会自动插入封面
- **报告标题**：设置封面标题的字体、字号、是否加粗、对齐方式
- **单位名称**：设置封面单位行的字体和字号
- **日期**：设置封面日期行的字体和字号

### 4.3 标题设置

支持一、二、三级标题独立设置：

| 设置项 | 说明 | 一级标题默认值 | 二级标题默认值 | 三级标题默认值 |
|--------|------|-------------|-------------|-------------|
| 字体 | 中文字体 | 黑体 | 黑体 | 仿宋 |
| 字号 | 中文字号 | 三号 | 四号 | 小四 |
| 加粗 | 是否加粗 | ✅ | ✅ | ✅ |
| 对齐 | 对齐方式 | 居中 | 左对齐 | 左对齐 |
| 段前距 | 标题前的间距 | 12pt | 8pt | 6pt |
| 段后距 | 标题后的间距 | 6pt | 4pt | 3pt |

### 4.4 正文设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 字体 | 仿宋 | 正文主体字体 |
| 字号 | 小四 | 正文主体字号 |
| 加粗 | 不加粗 | 正文一般不加粗 |
| 对齐 | 两端对齐 | 仿 Word 两端对齐效果 |
| 行距 | 1.5 | 1.0 / 1.25 / 1.5 / 2.0 可选 |
| 首行缩进 | 2字符 | 按当前字号自动换算实际宽度 |

### 4.5 表格设置

- **表头文字加粗**：第一行（表头行）的文字自动加粗
- **表头添加灰色底纹**：表头行添加浅蓝色底纹（颜色：D9E2F3）
- **单元格对齐**：所有单元格的文字对齐方式
- **自动调整表格宽度**：表格宽度自动适应页面

### 4.6 图片设置

- **图片居中对齐**：文档中的图片自动居中
- **最大宽度**：图片超过最大宽度时自动等比缩小（不会放大）

### 4.7 页眉/页码设置

**页眉设置：**
- 页眉内容：输入页眉文字
- 字体、字号、对齐方式
- 首页不显示页眉：勾选后，文档首页不显示页眉

**页码格式选项：**

| 格式 | 效果 |
|------|------|
| 1, 2, 3... | 纯数字页码 |
| -1-, -2-... | 横线包裹的数字页码 |
| 第1页 | 中文页码 |
| 第1页 / 共N页 | 中文页码+总页数 |

### 4.8 目录/落款设置

- **插入目录页**：勾选后，在文档最前面插入目录页
- **落款右对齐**：包含"盖章""公司"等关键词的段落自动右对齐
- **日期格式**：支持"年月日""点分""横线""斜线"四种格式

## 5. 模板管理

### 5.1 模板分类

| 分类 | 适用场景 |
|------|---------|
| 审计报告 | 审计业务报告 |
| 税审报告 | 税务审计报告 |
| 投标文件 | 招投标文件 |
| 通用模板 | 一般文档 |
| 其他 | 其他类型 |

### 5.2 导入模板

```
操作步骤：
1. 点击「📥 导入模板」
2. 选择一个已有格式的 Word 文档
3. 程序自动提取该文档的格式设置
4. 为模板命名并选择分类
5. 保存成功
```

> 💡 **提示**：导入模板时，程序会自动提取以下格式：
> - 标题样式（从文档样式和实际段落中提取）
> - 正文字体和字号（取出现最多的字体）
> - 页眉内容和格式
> - 表格格式
> - 落款格式

### 5.3 保存模板

手动调整好格式后，点击「💾 保存为模板」，选择分类即可永久保存。

### 5.4 使用模板

在左侧模板列表中点击模板名称，格式参数会自动加载到右侧各标签页。

### 5.5 模板存储位置

| 系统 | 路径 |
|------|------|
| Windows | `%APPDATA%\.底稿整理助手\templates\` |
| Mac/Linux | `~/.底稿整理助手/templates/` |
| PyInstaller 打包 | exe 所在目录 `.底稿整理助手\templates\` |

模板文件格式为 `.ftpl`（JSON 格式），可以用文本编辑器打开查看和修改。

## 6. 常见问题 FAQ

### Q1：应用模板后，目录和页码没有更新？

**A：** 这是正常的。目录和页码是 Word 的"域代码"，需要在 Word 中手动更新：
1. 打开调整后的文档
2. 按 `Ctrl + A`（全选）
3. 按 `F9`（更新域）
4. 选择"更新整个目录"→ 确定

### Q2：应用模板后文档格式不对？

**A：** 可能的原因和解决方法：
- **正文被错误应用了标题格式**：检查原文档的标题是否使用了"标题1/2/3"样式
- **图片位置变了**：检查"图片设置"中的居中和缩放选项
- **落款没有右对齐**：落款段落中需要包含"盖章""公司"等关键词才能被识别

### Q3：打开文件后提示"无法识别文档结构"？

**A：** 可能是文档的标题没有使用 Word 的"标题"样式。建议：
1. 在 Word 中，选中标题文字
2. 在"开始"选项卡中点击"标题1""标题2""标题3"样式
3. 保存后重新用底稿整理助手打开

### Q4：备份文件在哪里？

**A：** 备份文件在原文件同目录下，文件名格式为：`原文件名_备份_20260615_133000.docx`

### Q5：支持 .doc 格式吗？

**A：** 不支持。仅支持 `.docx` 格式（Office 2007 及以上版本）。如果是 `.doc` 格式，请先用 Word 另存为 `.docx`。

### Q6：应用模板时提示"文件被占用"？

**A：** 请先关闭 Word 中打开的该文件，然后重试。

### Q7：如何制作 exe 可执行文件？

**A：** 在安装了 Python 和依赖的 Windows 电脑上运行：
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "底稿整理助手" format_tool.py
```
生成的 exe 文件在 `dist` 目录下。
